# Do students exhibit different writing behaviors when told their work will be graded by AI?
DOI

This repository is the poster, code, and paper written for my URSA Engage research.

## Poster

Poster is written using [typst](https://typst.app/).

### Generating

OSU Fonts are required for proper generation. You can find them through the [OSU Branding page](https://communications.oregonstate.edu/brand-guide/typography)

```bash
# compile to pdf
typst compile main.typ

# watch during edits
typst watch file.typ
```
